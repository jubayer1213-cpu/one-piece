About This Project

This is a fan-made website template focused on One Piece and the Straw Hat Pirates. It uses official character images if you provide them. For this demo, placeholder images are included with exact file names — just replace them with official images you own the rights to use.

Hosting is easy: upload these files to a public GitHub repository and enable GitHub Pages.
